# trpg-toolkit

## Running the Test Suite

### 1. Clone the repository

```bash
git clone https://github.com/eblaug-uw/trpg-toolkit.git
```

### 2. Move into the frontend directory

```bash
cd trpg-toolkit/frontend
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the tests

```bash
npm test
```

Make sure you are inside the `frontend/` directory before running this command — it will not work from the project root.

---

### Additional test commands (optional)

| Command | Description |
|---|---|
| `npm test` | Run all tests once |
| `npm run test:watch` | Re-run tests on file save |
| `npm run test:ui` | Open the Vitest browser UI |
| `npm run coverage` | Generate a coverage report |
